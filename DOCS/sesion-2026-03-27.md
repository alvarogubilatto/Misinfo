## Gotchas y aprendizajes

- El trigger `handle_new_user` falla si no tiene `SET search_path = public` y referencia explícita `public.profiles`. Fix crítico para que Supabase pueda crear usuarios.
- Vite no recarga `.env` en caliente — siempre reiniciar el servidor tras editar las variables.
- `VITE_SUPABASE_URL` debe ser `https://xxx.supabase.co` (no la publishable key, sin trailing slash).
- En Google Cloud Console: el Redirect URI es `https://[project].supabase.co/auth/v1/callback`, no localhost.
- En Supabase Dashboard: Site URL y Redirect URLs deben ser `http://localhost:5173` sin barra final.
